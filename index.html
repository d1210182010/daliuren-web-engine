專業大六壬線上排盤系統

歡迎使用本站提供的大六壬線上排盤工具。大六壬與奇門遁甲、太乙神數並稱「三式」，是傳統中國古代最高層次的預測學之一。本系統專為現代命理研究者、實戰預測師以及大六壬愛好者打造，將傳統的 Python 算命引擎完美 Web 化，提供最精確、最視覺化的排盤體驗。

本站排盤系統核心特色

  • 精確天文曆法計算：內建 ELP2000-82 曆法模型，嚴格以太陽實際過宮（中氣交接）精確到分秒為基準來切換月將，確保大六壬起課數據準確無誤。
  • 獨家氣數透視功能：動態氣數透視，一鍵解析四課三傳中的生剋關係、下賊上/上剋下箭頭，並將落入「空亡」的地支陷阱視覺化，大幅提升實戰抓取發用的直覺。
  • 晝夜雙軌貴人演算法：打破傳統死板設定，前端支援自由切換「傳統卯酉分界」與「真實天文地平線（日出日落）」來判定晝夜貴人，完美考證各流派法則。
  • 結構化數據萃取 (AI 輔助)：內建一鍵複製排盤結構化數據功能，可無縫對接 ChatGPT、Claude 等大語言模型 (LLM)，輔助進行現代化的命理分析與解盤。
